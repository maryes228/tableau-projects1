# Tableau Project 1 – Superstore Sales Dashboard

##  Description
Ce projet présente un dashboard interactif réalisé avec **Tableau** à partir du dataset Superstore.  
L’objectif est de fournir une vue claire et dynamique des ventes et profits, par région, catégorie et localisation géographique.

##  Objectifs
- Mettre en avant les **KPI clés** (Total Sales, Profit).
- Visualiser les ventes par **région** et par **catégorie**.
- Analyser la répartition des ventes sur une **carte géographique**.
- Rendre le dashboard **interactif** grâce aux filtres et actions (clic sur une région → mise à jour des autres vues).

##  Fonctionnalités
- **KPI en Big Numbers** : chiffres formatés et colorés.
- **Carte interactive** : clic sur un État filtre les autres graphiques.
- **Sales by Region** : barres comparatives par région.
- **Sales by Category** : mise en évidence automatique de la catégorie dominante.
- **Filtres dynamiques** : navigation par région et catégorie.

##  Résultats
Ce dashboard permet d’identifier rapidement :
- Les régions les plus performantes.
- Les catégories de produits dominantes.
- Les tendances géographiques des ventes.
